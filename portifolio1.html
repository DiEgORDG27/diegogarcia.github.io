Algoritmos Básicos que ja usei para praticar

1. Soma de Dois Números

function somar(a, b) {
            return a + b;
            }

2. Verificar se Número é Par

function ehPar(num) {
            return num % 2 === 0;
            }

Uso: ehPar(4) retorna true

3. Encontrar o Maior Número

function maiorNumero(a, b) {
            if (a > b) {
            return a;
            } else {
            return b;
            }
            }

Uso: maiorNumero(7, 3) retorna 7
